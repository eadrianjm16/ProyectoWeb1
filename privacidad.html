Política de Privacidad

Esta Política de Privacidad describe cómo se recopila, utiliza y protege la información proporcionada por los usuarios en este sitio web. Este proyecto es exclusivamente académico y no tiene fines comerciales.

1. Responsable del tratamiento

El responsable de los datos recogidos en este sitio es el autor del proyecto, desarrollado en un contexto educativo para el módulo de Lenguaje de Marcas. Este sitio no pertenece a ninguna empresa real ni recopila datos con fines comerciales.

2. Datos que se recopilan

El único lugar donde se solicita información personal es en el formulario de contacto. Los datos que el usuario puede introducir son:

  • Nombre
  • Correo electrónico
  • Motivo de consulta
  • Preferencias seleccionadas (checkboxes y radiobuttons)
  • Mensaje escrito en el área de texto

Estos datos se introducen voluntariamente por el usuario y se emplean únicamente para fines demostrativos dentro del contexto del proyecto.

3. Finalidad del tratamiento

Los datos introducidos en el formulario no se almacenan de forma permanente ni se utilizan para contactar con el usuario. Su procesamiento es únicamente demostrativo, destinado a mostrar el funcionamiento de un formulario en un entorno educativo.

4. Conservación de los datos

Los datos no se guardan en bases de datos ni se transfieren a terceros. Una vez se abandona la página o se actualiza, la información introducida se pierde. El sitio no dispone de sistema alguno de almacenamiento, registro o seguimiento.

5. Uso de cookies

Este proyecto no utiliza cookies propias ni de terceros. No se realiza análisis de navegación ni se recogen datos estadísticos del usuario.

6. Enlaces externos

El sitio puede incluir enlaces hacia páginas externas, especialmente relacionadas con productos o información oficial de PlayStation. Estos enlaces redirigen fuera del proyecto y sus políticas de privacidad son ajenas a este sitio educativo.

7. Seguridad

Dado que este sitio no recopila ni almacena datos personales, no se requiere la implementación de sistemas avanzados de seguridad. Aun así, se recuerda a los usuarios evitar enviar información sensible a través de formularios en sitios sin cifrado o que no sean oficiales.

8. Derechos del usuario

Debido a que este proyecto no almacena datos de usuarios, no es necesario ejercer derechos de acceso, rectificación o eliminación. En un entorno real, estos derechos serían aplicables según la normativa vigente.

9. Carácter académico del sitio

Este sitio web es parte de un proyecto educativo y no debe interpretarse como un servicio real de recopilación de datos ni como un sitio oficial de la marca PlayStation o sus empresas asociadas.
